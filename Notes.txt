# Node.js Project Notes

## 1. Setup and Installation

### Clone the repository:
If you haven't already cloned the project, run the following command:

git clone githublink

### Install dependencies:
Run this command to install all required packages from `package.json`:

npm install

## 2. Running the Application

### Start the Node.js app:
Run the following command to start the application:

npm run dev

Alternatively, if you are using a tool like `nodemon` for live reloading during development:

node index.js
